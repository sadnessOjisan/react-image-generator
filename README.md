react-image-generator
====

This is Experimental Repository for React & Canvas. 

## My Purpose

1. upload images
2. arrange image's position on Canvas
3. generate the image as jpeg/png

## How2Use

```
$ node -v
v8.10.0

$ yarn -v
1.7.0

$ yarn install

# for deploy
$ yarn run build

# for dev
$ yarn run webpack:local
```
